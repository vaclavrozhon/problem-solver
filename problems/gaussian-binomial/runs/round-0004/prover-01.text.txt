Ideas
- Provide a fully detailed specialization of the q–Lucas theorem at q=ζ_N and use it to rigorously derive the residue-class decomposition of a_N(n)=f_n(ζ_N).
- From the decomposition, extract the Fibonacci diagonal sums to obtain, for each residue s mod N, a closed form g_s(x)=A_s F_{2x}+B_s F_{2x+1}+C_s F_{2x+2} and the uniform second-order recurrence g_s(x+2)=3 g_s(x+1)−g_s(x).
- Record explicit coefficients A_s,B_s,C_s for N=3,4 to audit the mechanism and verify formulas directly. Conclude nonperiodicity uniformly in N by a linear-algebra argument on the companion matrix.

Symmetric-to-standard reduction
For a≥b≥0 and q=v^2,
  \[\qbinom{a}{b}_v = v^{b(b−a)}\,[a;b]_q = q^{\frac{b(b−a)}2}\,[a;b]_q.\]
Proof. Using [m]_v=(v^m−v^{−m})/(v−v^{−1})=v^{1−m}(1−q^m)/(1−q), the product telescopes to v^{∑_{j=0}^{b-1}(1−(a−j))−∑_{j=1}^{b}(1−j)}=(v^2)^{b(b−a)/2}.
In particular, for a=n+i, b=2i,
  \[\qbinom{n+i}{2i}_v = q^{i(i−n)}\,[n+i;2i]_q,\]
so
  \[f_n(q)=\sum_{i=0}^n q^{i(i−n)}\,[n+i;2i]_q.\]

q–Lucas at a primitive N-th root (detailed proof)
Lemma. Let q=ζ_N. For integers a,b≥0, write a=a_1N+a_0, b=b_1N+b_0 with 0≤a_0,b_0<N. Then
- if b_0>a_0, [a;b]_q|_{q=ζ_N}=0;
- if b_0≤a_0, [a;b]_q|_{q=ζ_N}=\binom{a_1}{b_1}·[a_0;b_0]_q|_{q=ζ_N}.
Proof. Factor (q;q)_m=∏_{j=1}^m(1−q^j) as
  (q;q)_m = G(m)·H(m),
where G(m)=∏_{k=1}^{⌊m/N⌋}(1−q^{kN}) and H(m)=∏_{r=1}^{N−1}∏_{k=0}^{⌊(m−r)/N⌋}(1−q^{kN+r}). Then
  [a;b]_q = \frac{G(a)}{G(b)G(a−b)} · \frac{H(a)}{H(b)H(a−b)}.
At q=ζ_N, set z=q^N. Then G(m)=∏_{k=1}^{⌊m/N⌋}(1−z^k). The exponent of the simple zero at z=1 in G(a)/[G(b)G(a−b)] is c:=⌊a/N⌋−⌊b/N⌋−⌊(a−b)/N⌋, which equals 1 iff b_0>a_0 and 0 otherwise. Thus if b_0>a_0 the ratio vanishes and [a;b]_q(ζ_N)=0. If b_0≤a_0, then c=0 and
  lim_{z→1} \frac{G(a)}{G(b)G(a−b)} = \frac{(z;z)_{a_1}}{(z;z)_{b_1}(z;z)_{a_1−b_1}}\Big|_{z=1} = \binom{a_1}{b_1},
using lim_{z→1}(z;z)_m/(1−z)^m=m! (since 1−z^k=(1−z)(1+z+⋯+z^{k−1})→(1−z)·k).
For H, note that for m=m_1N+m_0,
  H(m)(ζ_N)=\prod_{r=1}^{N−1}(1−q^r)^{m_1} · \prod_{r=1}^{m_0}(1−q^r) = C_N^{m_1}·(q;q)_{m_0},
where C_N=∏_{r=1}^{N−1}(1−q^r)≠0. Therefore
  \frac{H(a)}{H(b)H(a−b)}(ζ_N)=\frac{(q;q)_{a_0}}{(q;q)_{b_0}(q;q)_{a_0−b_0}} = [a_0;b_0]_q(ζ_N).
Combining yields the claim. ∎

Fibonacci diagonal sums
Lemma. For x≥0 and δ,c∈{0,1}, define S_{δ,c}(x)=∑_{y≥0}\binom{x+y+c}{2y+δ}. Then
  S_{0,0}(x)=F_{2x+1}, S_{1,0}(x)=F_{2x}, S_{1,1}(x)=F_{2x+2}, S_{0,1}(x)=F_{2x+3}.
Proof. Set S_0(x)=S_{0,0}(x), S_1(x)=S_{1,0}(x). Pascal gives S_0(x+1)=S_0(x)+S_1(x), S_1(x+1)=S_1(x)+S_0(x), with S_0(0)=1,S_1(0)=0; hence S_0(x)=F_{2x+1}, S_1(x)=F_{2x}. The c=1 cases follow by S_{δ,1}(x)=S_{δ,0}(x+1).

Residue-class decomposition and recurrence
Theorem. Fix N≥1 and q=ζ_N. For n=xN+s with 0≤s<N,
  f_{xN+s}(q)=\sum_{r=0}^{N−1} q^{r(r−s)}\,[a_0(r);b_0(r)]_q · S_{δ(r),c(r)}(x),
where for i=yN+r we set c(r)=⌊(s+r)/N⌋, δ(r)=⌊2r/N⌋, a_0=s+r−c(r)N, b_0=2r−δ(r)N. In particular, there exist constants A_s,B_s,C_s∈ℚ(ζ_N) such that
  g_s(x):=f_{xN+s}(q)=A_s F_{2x}+B_s F_{2x+1}+C_s F_{2x+2}.
Moreover, the branch (δ=0,c=1) always vanishes since then b_0=2r and a_0=s+r−N, hence b_0−a_0=r−s+N>0, so [a_0;b_0]_q=0 by q–Lucas.
Proof. By the symmetric-to-standard reduction and the choice q^N=1, q^{i(i−(xN+s))}=q^{r(r−s)} depends only on r,s. Apply the q–Lucas lemma with a=xN+s+i and b=2i to factor each term. Sum over y≥0 and invoke the Fibonacci sums. ∎
Corollary. For each s, g_s(x) satisfies the linear recurrence
  g_s(x+2)=3g_s(x+1)−g_s(x) (x≥0).
This follows from F_{n+4}=3F_{n+2}−F_n and linearity.

Uniform nonperiodicity for fixed N
Proposition. For every N≥1, the sequence a_N(n)=f_n(ζ_N) is not periodic in n.
Proof. If a_N were P-periodic, then each residue subsequence g_s(x)=a_N(xN+s) would be periodic. Let M=[3 −1; 1 0]. The recurrence gives [g_s(x+1),g_s(x)]^T=M^x [g_s(1),g_s(0)]^T. Periodicity with period T implies M^T v=v for v=[g_s(1),g_s(0)]^T≠0. The eigenvalues of M are λ_{±}=(3±√5)/2, neither a root of unity; thus 1 is not an eigenvalue of M^T. Hence v=0, contradiction. ∎

Explicit subsequence at multiples of N
Proposition. For s=0,
  f_{xN}(ζ_N)=F_{2x+1}+C_N F_{2x},
where
  C_N=\sum_{r=\lceil N/2\rceil}^{N−1} ζ_N^{r^2}\,[r;2r−N]_q\big|_{q=ζ_N}.
Proof. For s=0, c(r)=0 for all r. The r=0 term contributes F_{2x+1}. For 1≤r≤⌊(N−1)/2⌋, b_0=2r>a_0=r so those terms vanish. For r≥⌈N/2⌉, δ=1 and each term contributes q^{r^2}[r;2r−N]_q F_{2x}. Summing yields the formula. ∎
This already forces nonperiodicity since g_0(x) is a nonzero solution of g(x+2)=3g(x+1)−g(x).

Auditable examples (coefficients A_s,B_s,C_s)
- N=3 (q=ζ_3): Using [2;1]_q=1+q and 1+q+q^2=0,
  • s=0: B_0=1, A_0=ζ_3(1+ζ_3)=−1, C_0=0, so a_3(3x)=F_{2x+1}−F_{2x}.
  • s=1: B_1=2, A_1=C_1=0, so a_3(3x+1)=2F_{2x+1}.
  • s=2: B_2=1, C_2=1, A_2=0, so a_3(3x+2)=F_{2x+1}+F_{2x+2}=F_{2x+3}.
Checks: a_3(0)=1, a_3(1)=2, a_3(2)=2, a_3(3)=1.
- N=4 (q=i): Using [3;2]_q=[3;1]_q=1+q+q^2=i and [2;1]_q=1+q,
  • s=0: B_0=1, A_0=0, C_0=0 ⇒ a_4(4x)=F_{2x+1}.
  • s=1: B_1=2, A_1=−1, C_1=0 ⇒ a_4(4x+1)=2F_{2x+1}−F_{2x}.
  • s=2: B_2=2, C_2=1, A_2=0 ⇒ a_4(4x+2)=2F_{2x+1}+F_{2x+2}.
  • s=3: B_3=1, C_3=0, A_3=0 ⇒ a_4(4x+3)=F_{2x+1}.
Direct computations for x=0,1 match these formulas.

Examples (quick counterexample)
- N=2: From the q=−1 specialization, a_2(2t+1)=F_{2t+3}, a_2(2t)=F_{2t−1} (with a_2(0)=1). This is unbounded, hence nonperiodic.
- N=1: f_n(1)=∑_{i=0}^n \binom{n+i}{2i}=F_{2n+1}, also nonperiodic.

Obstacles
- The q–Lucas proof at ζ_N is notationally heavy if done entirely inside ℤ[q]/(Φ_N), but the G–H factorization with the z=q^N limit cleanly separates the vanishing multiples-of-N factors and yields a short, rigorous argument.
