#### Abstract

### 1. Nowhere‚Äëzero flows for infinite graphs

**Definition 1 (weak flow).**  
Given an orientation \(D\) of a graph \(G\) and an abelian group \(A\), a **weak \(A\)‚Äëflow** is a function \(\varphi:E(G)\to A\) such that **Kirchhoff‚Äôs law for each vertex cut** holds:
\[
\forall v\in V(G):\quad \sum_{e\in \delta^-(v)}\varphi(e)=\sum_{e\in \delta^+(v)}\varphi(e).
\]

**Definition 2 (strong flow).**  
Given \(D,G,A\) as above, a **strong \(A\)‚Äëflow** is a function \(\varphi:E(G)\to A\) such that **Kirchhoff‚Äôs law for each cut** holds:
\[
\forall S\subseteq V(G):\quad \sum_{e\in \delta^+(S)}\varphi(e)=\sum_{e\in \delta^-(S)}\varphi(e).
\]

In either case, \(\varphi\) is a **nowhere‚Äëzero** weak/strong \(A\)‚Äëflow if \(\varphi^{-1}(0)=\varnothing\).

**Definition 3 (bridge).**  
An edge \(e\) of \(G\) is a **bridge** if deleting \(e\) increases the number of connected components. A bridge is **2‚Äëway infinite** if both resulting components are infinite, and **1‚Äëway infinite** otherwise.

Every strong flow is a weak flow. While the two notions are equivalent on finite graphs, they diverge on infinite graphs: e.g., on a two‚Äëway infinite path, a nowhere‚Äëzero weak \(A\)‚Äëflow exists but a strong one does not. In [6] the authors construct a 3‚Äëconnected graph with two ends that admits a nowhere‚Äëzero weak \(A\)‚Äëflow but no nowhere‚Äëzero strong \(A\)‚Äëflow. Nearly all finite flow theorems extend to infinite **strong** \(A\)‚Äëflows via topological compactness; here we use **logical** compactness to obtain:

**Theorem 4.**  
*A countably infinite, locally finite graph without a 1‚Äëway infinite bridge has a nowhere‚Äëzero weak 6‚Äëflow.*

**Definition 9 (space of rooted graphs).**

- For all \(D>2\), \(\mathbf{G}_{\bullet,D}\) is not meagre.

### Main results on typical properties

**Theorem 14.**  
*The set of bridgeless graphs is nowhere dense (i) in \(\mathbf{G}_{\bullet,\le D}\) for all \(D>2\); and (ii) in \(\mathbf{G}_{\bullet,=D}\) for all odd \(D>2\).*  
In D‚Äëregular graphs with even \(D\), 1‚Äëway infinite bridges cannot exist; instead we obtain:

**Theorem 15.**  
*The set of graphs without a cut vertex is nowhere dense in \(\mathbf{G}_{\bullet,D}\) for all \(D>2\).*

**Theorem 16.**  
*The set of graphs with a 2‚Äëway infinite bridge is meagre (but not nowhere dense) in \(\mathbf{G}_{\bullet,D}\) for all \(D>2\).*

We also considered other (a)typical properties:

**Theorem 17.**  
*The set of graphs with edge‚Äëchromatic number \(D\) is nowhere dense in \(\mathbf{G}_{\bullet,D}\) for all \(D>2\).*

### 3. Proof sketches

**Proof of Theorem 4 (sketch).**  
Let the vertices of \(G\) be \(v_1,v_2,\ldots\) and orient each edge \(\{v_i,v_j\}\) from the smaller to the larger index. For each radius \(r\), augment the r‚Äëball \(B_r(G,v_1)\) to a finite multigraph \(G_r\) by adding a sink vertex \(v_\infty\) and connecting boundary edges to \(v_\infty\). Introduce Boolean variables \(F_{e,1},\dots,F_{e,6}\) encoding which of six nonzero values an edge \(e\) receives, enforce exclusivity \(\bigvee_i F_{e,i}\) and \(\bigwedge_{j\ne i}\neg F_{e,j}\), and write linear parity constraints at each vertex (Kirchhoff‚Äôs law). Let \(F_{G_r}\) be the conjunction over vertices of \(G_r\); then every finite subset of the constraint family \(\{F_{G_r}\}_r\cup\{F_{e,i}\}_e\) is satisfiable (by [7]), hence by compactness [4] the whole set is satisfiable, yielding a nowhere‚Äëzero weak 6‚Äëflow on \(G\).

**Proof idea for Theorem 14.**  
Given an r‚Äëball \(F\), construct an r‚Ä≤‚Äëball \(F'\) such that every extension of \(F'\) contains a bridge. For \(\mathbf{G}_{\bullet,\le D}\), attach a leaf to a boundary vertex of degree \<\(D\). For \(\mathbf{G}_{\bullet,=D}\) handle regularity via parity gadgets: define \(S(F)=\sum_{v\in L_r}(D-\deg v)\) where \(L_r\) is the boundary; use \(K_{D+1}-e\) gadgets and tree‚Äëlike expansions to force parity \(S(F)=1\), then attach a new vertex to create a bridge. Hence the bridgeless set is nowhere dense.

**Proof idea for Theorem 15.**  
In even \(D\)‚Äëregular graphs, use gadgets to force a cut vertex; in odd \(D\) or degree‚Äëbounded spaces, this also follows from the previous construction.

**Proof idea for Theorem 16.**  
Being ‚Äúthere is a 2‚Äëway infinite bridge at distance \(k\)‚Äù is nowhere dense; take a union over \(k\in\mathbb{N}\) to get a meagre (but not nowhere‚Äëdense) set.

**Proof idea for Theorem 17.**  
For each \(D\), there exist \(D\)‚Äëregular graphs with \(\chi'(G)=D+1\) (even \(D\): complete graphs or any odd‚Äëorder \(D\)‚Äëregular graph; odd \(D\): by Tutte‚Äôs theorem). Removing an edge preserves \(\chi'=D+1\) by parity. Embed parity gadgets within r‚Äëballs so that every extension has edge‚Äëchromatic number \(D+1\), hence the set is nowhere dense.

### 4. Future work (the conjectures)

**Question 18.** *Is the fact that a property is nowhere dense or meagre independent of the root?*  
A change of root gives one type of change of metric; however, many metrics on the space of countable graphs are possible.

**Question 19.** *Is there another metric on \(\mathbf{G}_{\bullet,D}\) where typical properties are different?*  
For instance, is there a metric in which **bridgeless graphs are typical**?

We also aim to investigate additional (a)typical properties in the spirit of Theorem 17.
